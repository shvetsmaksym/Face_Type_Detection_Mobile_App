# Face_Type_Detection_Mobile_App
## Krótki opis
Aplikacja wykrywająca typ twarzy i dopasowująca odpowiedni typ opraw.

## Wykorzystane technologie
1. Python 3.7.0
2. kivy 2.0.0
3. tensorflow 2.6.2
4. numpy 1.19.5
5. matplotlib 3.3.4
6. scipy 1.5.4
7. opencv-python 4.5.5.64
